# Enterprise-Network-Infrastructure-Design
## Overview
A scalable client/server network infrastructure designed to support business growth, with comprehensive service configuration and robust security controls.

## Project Highlights

- Designed a scalable client/server network supporting projected growth from 20 to 50 users.
- Configured file, print, email, and web services; implemented wired and wireless connectivity with security controls.
- Developed network topology diagrams using Cisco Packet Tracer.

## Tools & Technologies

- **Cisco Packet Tracer** – Network topology design and simulation
- **Client/Server Architecture** – Scalable infrastructure planning
- **Network Services** – File, print, email, and web service configuration
- **Security Controls** – Wired and wireless network security implementation
